"One of a series of databases created, designed, and built with the GK Collabs Core Six™ concept.

Built in a Linux (Debian) environment, this toolkit automates the CRUD (Create, Read, Update, Delete) cycle for career management and also features search and summary functions."

"The DVD Archive is an advanced media management tool that utilizes SQL aggregation to provide deep insights into a film collection. This folder features the 'DVD Dashboard,' which generates automated summaries categorized by both Film Director and Genre, demonstrating complex data grouping and professional reporting logic."
